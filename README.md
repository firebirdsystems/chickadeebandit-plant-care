# Plant Care

A [Chickadee Bandit](http://chickadeebandit.com) app.

Track watering, fertilizing, and any other care for your household plants. Color-coded urgency shows what needs attention. One tap to log it.

## Features

- Configure any care activities per plant with custom intervals
- Color-coded urgency: green (on track), amber (due soon), red (overdue)
- One-tap logging with optional notes
- Full care history per plant and activity
- Supports multiple plants

## Install

In your hub, go to **Apps → Install from URL** and paste:

```
https://github.com/firebirdsystems/chickadeebandit-plant-care/releases/latest/download/bundle.json
```
